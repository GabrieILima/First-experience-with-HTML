# 🌐 First Experience with HTML

Primeiro contato com HTML, desenvolvido durante a Formação HTML Web Developer na [DIO.me](https://dio.me).

---

## 📌 Sobre o projeto

Página web construída para praticar os fundamentos do HTML, explorando a estrutura básica de documentos, principais tags e formatação de conteúdo.

Conceitos praticados:
- Estrutura básica de um documento HTML (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Tags de texto (`<h1>`–`<h6>`, `<p>`, `<strong>`, `<em>`)
- Listas ordenadas e não ordenadas (`<ol>`, `<ul>`, `<li>`)
- Links e âncoras (`<a>`)
- Inserção de imagens (`<img>`)
